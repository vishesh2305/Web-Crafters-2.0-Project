Learn HTML

Introduction to HTML

HTML (HyperText Markup Language) is the standard language for creating web pages.

<!DOCTYPE html>
<html>
  <head>
    <title>My First Page</title>
  </head>
  <body>
    <h1>Hello, World!</h1>
  </body>
</html>

Basic HTML Tags

  • <p> - Defines a paragraph.
  • <h1> to <h6> - Define headings.
  • <a> - Defines hyperlinks.
  • <img> - Defines images.

HTML Lists

HTML supports ordered (<ol>) and unordered (<ul>) lists.

<ul>
  <li>Item 1</li>
  <li>Item 2</li>
</ul>

HTML Tables

Tables are created using the <table> tag.

<table border="1">
  <tr>
    <th>Name</th>
    <th>Age</th>
  </tr>
  <tr>
    <td>Alice</td>
    <td>25</td>
  </tr>
</table>
